# UAS-Net: Uncertainty-Aware Semi-Supervised Multimodal Survival Network

This repository contains the official PyTorch implementation of the paper:  
**"UAS-Net: Uncertainty-Aware Semi-Supervised Multimodal Survival Network for Esophageal Cancer Prognosis Prediction"**.

## 🚀 Overview
Accurate survival risk prediction is essential for personalized treatment. UAS-Net integrates 3D Masked Autoencoder (3D-MAE) pre-training, an asymmetric multimodal feature fusion mechanism, and an uncertainty-aware Mean Teacher framework to address the "high-dimensional, small-sample" dilemma in medical survival analysis.
